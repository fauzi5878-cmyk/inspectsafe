INSPECTSAFE V3.1 — MASTER PIC + WHATSAPP

New functions:
- Master PIC dropdown
- Automatic PIC WhatsApp number
- Optional WhatsApp group selector
- Unique sequential Finding No: HSE-YYYY-0001
- Finding number stored in Firestore and reports

IMPORTANT SETUP:
1. Open master-data.js using Notepad.
2. Enter each PIC phone number in Malaysia format: 60123456789
3. Paste WhatsApp group invite/open links where required.
4. Upload all files to the root of the GitHub repository.
5. Replace existing files and Commit changes.

WhatsApp groups:
A normal web app cannot automatically place a prepared message inside a group.
InspectSafe copies the message, opens the group link, and the user pastes and sends it.

Security:
Firestore is currently public for testing. Authentication and secure rules must be added before full production use.
