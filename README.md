# InspectSafe v2.0 Alpha

Digital HSE inspection and reporting prototype.

## Current features
- Dashboard
- New inspection form
- Photo attachment
- Local browser storage
- WhatsApp message to PIC
- Workplace Inspection Report
- SHO Monthly Summary
- Print / Save as PDF

## Deploy
Upload all files to the root of the GitHub repository:
- index.html
- styles.css
- app.js
